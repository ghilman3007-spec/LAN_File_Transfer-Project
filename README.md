A Basic File Transfer Application to send files on a Local Area Network with Built-in Compression using java GZIP compression class.

Usage:

1. Select the role of your device: Sender or Receiver
2. Connect to the other device you want the files to send to
3. Select the file you want to send

NOTE: make sure to allow the Windows Defender to connect to InBound and OutBound Traffic, if the 
      Pop-Up does not appear, then manually allow your device to receive and send data on private network.
